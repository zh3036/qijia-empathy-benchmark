# 齐家 · 共情评估基准（志愿者协作）

帮我们打磨一把更准的尺子——用来衡量 AI 教练的**共情**。

这是一个开放的志愿者协作仓库。我们在做一件事：评估**心理教练/咨询对话**的质量——尤其是**共情**。我们有量表、有对话、有 36 位评分者打的分。首轮数据显示，这套**评分方法**还不够灵敏，需要先校准，才能用它可靠地衡量和改进 AI 教练。我们需要你帮忙校准它。

> **关于齐家 AI：** 齐家AI 是一款面向中国家长的**免费、公益、开源**的 AI 家庭教练，由智能向善社会创新网络（注册于杭州西湖区的非营利组织）开发，参与者全部是志愿者。本仓库是齐家 AI 评估工作的一部分。
> 🔗 官网：<https://westlakeaiforgood.com/>　·　完整介绍与常见问题见 [docs/关于齐家AI.md](docs/关于齐家AI.md)

## 为什么需要你

「共情」是一种很主观的品质，要从文字对话里稳定地给它打分本身就很难——这是心理学和 AI 评估领域公认的挑战。我们让 **36 位评分者**用一套标准量表（TES 共情量表，9 个题目，1–7 分）给 **20 段 AI 教练对话**打了分，希望用它来持续衡量、改进教练的共情表现。首轮分析显示，目前这套**评分方法**还有几个可以改进的地方：

- **不同评分者打分差异较大。** 同一段对话，分数高低很大程度上取决于「谁在打分」——评分者差异占了总方差的 **54%**。
- **量表的评分者一致性还偏低**（ICC 接近 0）。这在「凭文字给共情打分」时很常见，说明量表的评分标准还需要更清晰。
- **分数区分度不足。** 20 段对话的整体共情均值都落在 **4.3–5.7**（满分 7），分布偏集中，目前还不太能拉开「更好」与「有待改进」的差距。

> 这些是**评分方法**当前的局限，并不代表 AI 教练本身的水平——恰恰相反，正因为我们想认真地评估并提升它，才要先把这把「尺子」校准。详细分析见在线报告：<https://review-deploy-indol.vercel.app/>（含 TES 锚定参考页 <https://review-deploy-indol.vercel.app/anchor/>）。

把这套评分方法校准好，需要两样东西——也正是你能帮上忙的地方：

| 任务 | 一句话 | 入口 |
|---|---|---|
| **① 共情锚定示例** | 为每个共情题目，找出“1 分长什么样、7 分长什么样”的对话例子 | [`tasks/01-anchor-book/`](tasks/01-anchor-book/README.md) |
| **② 补充极端对话** | 找/写一些**明显很好**或**明显很差**的新对话，把分数差距拉开 | [`tasks/02-new-dialogs/`](tasks/02-new-dialogs/README.md) |

> 这两件事就像给尺子刻上清晰的刻度（任务一），再找来又长又短的东西去量（任务二）。

## 怎么参与

**不需要写代码**，全程可以在 GitHub 网页上完成：

1. 读 [`tasks/`](tasks/) 里你想做的任务说明。
2. 在 [`data/`](data/README.md) 里翻对话和分数，找素材。
3. Fork 本仓库 → 编辑对应文件 → 提一个 **Pull Request**。

详细步骤见 **[CONTRIBUTING.md](CONTRIBUTING.md)**。
