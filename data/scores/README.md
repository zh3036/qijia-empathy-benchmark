# 评分数据（共情 / TES）

来自 36 位人工评分者对 20 段对话的共情（TES）评分，1–7 分。

## 文件

| 文件 | 内容 |
|---|---|
| [`empathy-by-dialog.md`](empathy-by-dialog.md) | **聚合表**（可读）：逐对话 × 逐题的截尾均值，按整体均值排序。 |
| `empathy-by-dialog.csv` | 同上的 CSV，含均值、截尾均值、有效人数 `n`。 |
| `empathy-raw.csv` | **原始逐人评分**：每行一个 `(评分者, 对话, 题目, 分数)`。 |

`empathy-raw.csv` 列：`rater_id, dialog_name, dialog_id, item, score`。

## 怎么算的

- **截尾均值**：对每题，去掉最高和最低各 10% 的评分后取平均，降低极端评分者的影响。
- 只统计 1–7 的有效数字评分；空白、“未填写”、自由文本（`evaluation_notes`）不计入。

## 重要提醒：这些分数本身就不太可靠

这正是本项目要解决的问题：

- **评分者偏差占总方差的 54%**——同一段对话，换个人打分，结果可能差很多。
- **共情量表 ICC ≈ 0**——评分者之间几乎没有一致性。

**所以：** 把这些分数当作“粗略参考”，别当作“真值”。它们能帮你大致看出哪些对话偏高、哪些偏低（36 人平均后的**排名**还算稳定），但单看某一格的具体数值意义有限。你的任务正是帮我们把这把尺子变准。
