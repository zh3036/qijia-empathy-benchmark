# 任务二 · 补充极端对话

**找来又好又差的对话，把分数差距拉开。**

## 为什么

现有 20 段对话的整体共情均值都落在 **4.3–5.7**（满分 7）——没有明显的高分（≥6）或低分（≤4）样本。一个评估基准如果只有“中等”样本，就没法检验尺子能不能区分优劣。我们需要补充：

- **明显很好**的对话（共情做得非常到位）
- **明显很差**的对话（共情明显缺失：冷漠、说教、答非所问、忽视情绪）

> - 这些新对话**不需要预先打精确分数**——只要你判断它“明显好”或“明显差”即可。
> - 尽量别和任务一的锚定示例、或现有 20 段对话用同一段。目前提交还很少，重复的概率不大，你只要避开明显的 20 段即可；已提交的对话会记在 [`submissions/index.md`](submissions/index.md)。

## 你要做的事

1. 找一段对话。三种来源都可以：
   - **从素材库挑**：[`../../data/low-rating-pool/`](../../data/low-rating-pool/index.md) 有 109 段 AI 评分偏低的对话，排在前面的共情很差，是现成的“差”样本。挑一段，在提交里注明来源文件。
   - **自己写**：编一段你认为典型的好/差教练对话。
   - **真实对话**：⚠️ 必须先**脱敏**，去掉一切真实姓名和可识别信息。
2. 在 [`submissions/`](submissions/) 下**新建一个文件**，文件名以倾向开头：`good-情绪命名.md` 或 `bad-冷漠说教.md`（文件名用中文没问题）。
3. 按 [TEMPLATE.md](TEMPLATE.md) 填写，可参考 [`submissions/EXAMPLE.md`](submissions/EXAMPLE.md)。
4. 提 PR（见 [../../CONTRIBUTING.md](../../CONTRIBUTING.md)）。

## 怎样算一个好的提交

- **倾向明确**：清楚标注是“明显好”还是“明显差”，并一句话说明理由。
- **聚焦共情**：好坏主要体现在共情上（理解、命名、接纳情绪 vs 冷漠、说教、忽视）。
- **来源清楚**：挑自素材库的注明文件名；自己写的注明“原创”；真实改编的确认已脱敏。
- **长度适中**：一般 5–12 轮。
