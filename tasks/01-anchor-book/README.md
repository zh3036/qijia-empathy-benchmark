# 任务一 · 共情锚定示例（Anchor Book）

**给共情这把尺子刻上清晰的刻度。**

## 这是什么

「锚定示例」就是一本参考册，规定**每个分数长什么样**：1 分的对话什么样、4 分什么样、7 分什么样，各配一段真实对话摘录。有了它，不同评分者对“几分”才有共同的标准——这正是提升评分者一致性（首轮 ICC 接近 0，还有很大改进空间）的关键一步。

我们为共情（TES）的 **9 个题目**各建了一个文件，每个文件有 7 个空槽（1–7 分各一），等你来填。

## 你要做的事

1. 打开 [`anchors/`](anchors/index.md)，选一个题目文件，例如 [`anchors/q4-warmth.md`](anchors/q4-warmth.md)。
2. 读题目描述，理解这道题在评什么（也见 [`../../data/questionnaire.md`](../../data/questionnaire.md)）。
3. 翻对话找一段**清楚体现某个分数**的摘录（来源见下）。
4. 把该分数下的 `_（待补充）_` 替换成你的例子（格式见 [TEMPLATE.md](TEMPLATE.md)）。
5. 提 PR（见 [../../CONTRIBUTING.md](../../CONTRIBUTING.md)）。

## 从哪里找素材

- **中间分（3–6 分）** —— 用 [`../../data/dialogs/`](../../data/dialogs/index.md) 的 20 段对话。它们的整体共情均值都在 **4.3–5.7**，是这些分数最现实的来源。看 [`empathy-by-dialog.md`](../../data/scores/empathy-by-dialog.md) 找逐题偏高/偏低的对话。
- **低分（1–2 分）** —— 20 段对话里几乎没有这么低的。可去 [`../../data/low-rating-pool/`](../../data/low-rating-pool/index.md)（按共情从差到好排序，最前面的最差）找冷漠/说教/忽视情绪的片段。⚠️ 那里的 AI 分数不是 1–7 量尺，请自己按题目描述判断该给几分。
- **高分（7 分）** —— 现有数据里基本没有。

> ⭐ **最缺 1、2、7 分的例子**（文件里标了星号）。但这正是因为现有对话**给不出**这些极端样本。所以对 1/2/7 分槽，你可以：
> - 从 `low-rating-pool` 找接近 1–2 分的片段；或
> - 自己**写**一小段示意性的 1 分 / 7 分摘录（请在理由里注明“示意”）；或
> - 提一个 PR 说明“在现有数据里找不到合适的 X 分例子”——这本身也是有用的信息。
>
> 3–6 分槽则优先从现有 20 段对话里选真实片段填。

## 怎样算一个好的锚定示例

- **2–3 轮就够**，截取最能说明问题的那几句。
- **典型、无歧义**——一眼就能看出为什么是这个分数。
- **注明来源**：哪段对话、第几轮。
- **一句话理由**：点出关键特征（如“直接命名并肯定了来访者的情绪”）。

填写格式见 **[TEMPLATE.md](TEMPLATE.md)**。
